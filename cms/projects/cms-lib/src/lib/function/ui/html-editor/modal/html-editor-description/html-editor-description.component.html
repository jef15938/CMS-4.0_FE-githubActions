1. 粗體

將框住的字體變粗體

2. 斜體

將框住的字體變斜體

3. 底線

將框住的字體加底線

4. 置中對齊

將框住的字置中

5. 左右對齊

將非最後一行的文字左右對齊

6. 靠左對齊

將框住的字靠左

7. 靠右對齊

將框住的字靠右

8. 條列項目

將該行變為條列項目

9. 插入子清單

注意：此功能為插入清單，若要在同一清單加入更多項目僅需按Enter即可。

10. 向外縮排

將框住的字往右縮排

11. 向內縮排

將框住的字往左縮排

12. 標記文字

將框住的字體變標註顏色或取消標記

13. 將選取文字/圖片增加連結:

  • 13-1:開啟方式: 選擇連結是否另開分頁
  • 13-2:連結方式: 選擇連結是站外連結或站內節點
  • 13-3:連結文字: 連結顯示的文字
  • 13-4:站外
  • 13-4-1:站外連結: 站外連結網址
  • 13-5:站內
  • 13-5-1:網站: 選擇網站
  • 13-5-2:節點: 選擇網站內的節點

14. 插入檔案: 開啟媒體庫選擇檔案

15. 插入圖片: 開啟媒體庫選擇圖片

16. 插入表格:

  • 16-1:行數: 欲插入表格行數
    • 16-2:列數: 欲插入表格列數
  • 注意：表格行列皆不能超過15行或列。

17. 插入Youtube影片:

  • 17-1:連結網址: 欲插入影片的連結網址

18. 表格功能：

  • 18-1:列: 列相關操作
  • 18-1-1:上方列: 在最後選取的表格上方插入一列
  • 18-1-2:下方列: 在最後選取的表格下方插入一列
  • 18-1-3:刪除列: 刪除最後選取的表格所在之列
  • 18-2:欄: 欄相關操作
  • 18-2-1:左側欄: 在最後選取的表格左側插入一欄
  • 18-2-2:右側欄: 在最後選取的表格右側插入一欄
  • 18-2-3:刪除欄: 刪除最後選取的表格所在之欄
  • 18-3:合併: 將選取的表格合併
  • 18-4:分割: 分割相關操作
  • 18-4-1:水平分割: 將最後選取的表格水平分割為兩格
  • 18-4-2:垂直分割: 將最後選取的表格垂直分割為兩格
  • 18-5:刪除表格: 刪除最後選取的該表格

19. 表格樣式：

  • 19-1:滿版縮放: 表格寬度設為100%等比例縮放
  • 19-2:Scroll: 寬度為固定pixel，超出時可左右滑動
  • 19-3:單筆顯示: 小版時單筆顯示
