1. 粗體

將框住的字體變粗體

2. 斜體

將框住的字體變斜體

3. 底線

將框住的字體加底線

4. 靠左對齊

將框住的字靠左

5. 置中對齊

將框住的字置中

6. 靠右對齊

將框住的字靠右

7. 條列項目

將該行變為條列項目

8. 插入子清單

注意：此功能為插入清單，若要在同一清單加入更多項目僅需按Enter即可。

9. 將選取文字/圖片增加連結:

  • 9-1:連結文字: 連結顯示的文字
  • 9-2:連結網址: 目前連結的網址
  • 9-3:開啟方式: 選擇連結是否另開分頁
  • 9-4:儲存變更: 儲存變更結果

10. 向外縮排

將框住的字往右縮排

11. 向內縮排

將框住的字往左縮排

12. 插入表格:

  • 12-1:行數: 欲插入表格行數
    • 12-2:列數: 欲插入表格列數
  • 注意：表格行列皆不能超過15行或列。

13. 標記文字

將框住的字體變標註顏色

14. 取消標記

將框住的字體取消標註顏色

15. 插入檔案: 請選擇檔案類型,初始會呈現最新10個檔案

  • 15-1:更多檔案: 按最下方的更多檔案來載入更多圖片(一次10張)
  • 15-2:搜尋功能: 利用關鍵字,搜尋媒體庫圖片

16. 插入Youtube影片:

  • 16-1:連結網址: 欲插入影片的連結網址
  • 16-2:框架高度: 由css控制
  • 16-3:儲存變更: 儲存變更結果

17. 表格功能：

  • 17-1:+上方列: 在最後選取的表格上方插入一列
  • 17-2:+下方列: 在最後選取的表格下方插入一列
  • 17-3:+左側欄: 在最後選取的表格左側插入一欄
  • 17-4:+右側欄: 在最後選取的表格右側插入一欄
  • 17-5:刪除欄: 刪除最後選取的表格所在之欄
  • 17-6:刪除列: 刪除最後選取的表格所在之列
  • 17-7:水平分割: 將最後選取的表格水平分割為兩格
  • 17-8:垂直分割: 將最後選取的表格垂直分割為兩格
  • 17-8:合併: 將目前選取的所有表格合併為同一格
  • 17-8:標記/取消: (標記上色/取消)目前所選取的表格
  • 17-8:刪除表格: 刪除最後選取的該表格

18. 表格樣式：

  • 18-1:滿版縮放: 表格寬度設為100%等比例縮放
  • 18-2:Scroll: 寬度為固定pixel，超出時可左右滑動
  • 18-3:收合: 當寬度超過時會自動收合超出的欄位
  • 18-4:單筆顯示: 小版時單筆顯示
